# PCB design, iteration 1
## Requirements
  * Breakout board for Teensy 3.6
  * Connectors
    * Midi in/out
    * TFT display
    * rotary encoder / switch
    * possibly use neopixels rings as indicator for rotary encoder.
  * 2 tactile switches (play/stop, record enabled)
